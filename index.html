What is this?
This is a static media host for rides (stories + reels) and garage (vehicles + tours). It exposes a machine-readable manifest.json that other sites can consume, and a simple browser UI for quick discovery.
/manifest.json Generated index of all media files with sizes, timestamps, and categories.
/api.json Human + machine-friendly description of the available endpoints.
/media/garage/* Garage photos/videos (e.g., vehicles, tours). No bundling in the main Nuxt repo.
/media/* All raw files (stories/reels/profile/garage). Served with long-term caching.
